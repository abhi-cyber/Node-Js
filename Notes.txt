Important points:-

1. Node is used to build server side app.
2. When working on Node you have no access to browser API's like DOM, Window, etc.
3. Unlike other frameworks, in Node we are not required to update our node in existing projects which save our app from crashing.
4. Node evaluates your code using either REPL (Read Eval Print Loop) or CLI (MAIN).
5. To run a js file using node, write the following command ==> node <name of the file>.
6. Node uses CommonJS library and its every file is module by default.

# GLOBAL Variables

1. __dirname ==> path to current directory
2. __filename ==> file name
3. require ==> frunction to use modules (CommonJS)
4. module ==> info about current modules (file)
5. process ==> info about env where the programis being executed

Note :-- To use them you can just console.log(<type the command here>)
